# Medical Appointment System

This project is a frontend application for managing medical appointments. It allows users to sign up, log in, book appointments, and manage their profiles.

## Project Structure

```
medical-appointment-frontend
├── src
│   ├── index.html          # Main entry point of the application
│   ├── signup.html         # User signup page
│   ├── login.html          # User login page
│   ├── book-appointment.html # Appointment booking page
│   ├── dashboard.html       # User dashboard displaying appointments
│   ├── profile.html         # User profile management page
│   ├── doctors.html         # List of available doctors
│   ├── settings.html        # User account settings page
│   ├── css
│   │   ├── styles.css       # Main styles for the application
│   │   ├── dashboard.css     # Styles specific to the dashboard
│   │   └── forms.css        # Styles for forms
│   └── js
│       └── main.js         # JavaScript for handling interactions
└── README.md               # Project documentation
```

## Features

- User authentication (signup and login)
- Appointment booking with doctor selection
- User dashboard to view and manage appointments
- Profile management for users
- List of available doctors with specialties
- Account settings for user preferences

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd medical-appointment-frontend
   ```

3. Open the `index.html` file in a web browser to view the application.

## Usage

- Users can create an account or log in to access their dashboard.
- Appointments can be booked by selecting a doctor and specifying a date and time.
- Users can view, edit, or cancel their appointments from the dashboard.
- Profile and settings pages allow users to manage their personal information and preferences.
